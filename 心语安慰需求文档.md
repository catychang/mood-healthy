# 心语安慰（Heart Verses）需求文档

## 1. 概述
本网站旨在通过用户选择情绪词或输入一段文字，提供简单、直接的鼓励话语，并允许用户收藏喜欢的话语和查看历史记录。整体设计采用深蓝黑色暗黑风格，营造出宁静、优雅的氛围。系统具备AI自主学习能力，根据用户收藏的内容优化和个性化推荐。

## 2. 目标用户
群体：所有希望快速获得情绪支持的人，尤其是那些感到压力、焦虑、疲劳、困惑等情绪波动的用户。

用户需求：用户希望通过简单的方式选择当前的情绪状态或输入一段文字，并获得鼓励或安慰的话语，同时能够收藏喜欢的内容和查看历史记录。

## 3. 功能需求

### 3.1 导航标签页
功能：在页面顶部提供三个主要标签页，方便用户在不同功能之间切换。

标签页：
- **我现在的心情**：主页，用户可以选择情绪或输入文字，获取鼓励话语。
- **我的收藏**：显示用户收藏的所有鼓励话语。
- **历史记录**：显示用户最近选择的情绪和对应的鼓励话语。

### 3.2 网站标题和介绍
功能：向用户说明网站的功能和用途。

内容：
- 网站标题：大标题文本，居中显示，如"Heart Verses"或"心语安慰"。
- 简短描述：一段简短文字，说明网站功能，如"选择你今天的感受，我们将提供安慰和指导的话语"。

### 3.3 情绪选择模块（我现在的心情页面）
功能：用户可以选择一个情绪词，系统根据选择提供对应的鼓励或安慰话语。

实现方式：
- 情绪词以网格卡片形式排列，每行显示4-5个情绪卡片。
- 卡片设计为深色背景，浅色文字，选中后突出显示（如参考图中的"Hope"卡片）。

情绪选项：
- 喜悦 (积极愉快的心情)
- 忧伤 (含蓄的悲伤表达)
- 担忧 (日常生活中的常见情绪)
- 烦躁 (常见的负面情绪表达)
- 安宁 (追求的平静心境)
- 向往 (希望与憧憬的情绪)
- 思念 (富有诗意的情感表达)
- 感动 (被他人善举或美好事物触动)
- 迷茫 (现代人常见的心理状态)
- 无奈 (面对生活压力时的复杂情绪)

### 3.4 自定义情绪词功能
功能：允许用户添加自己喜欢或常用的情绪词，补充默认情绪列表。

实现方式：
- 在情绪选择区域末尾添加一个"+"卡片，用户点击后可以输入自定义情绪词。
- 输入完成后，新的情绪词将作为卡片添加到情绪选择区域，与默认情绪卡片风格一致。
- 自定义情绪词卡片上可以有一个小的"×"图标，用户点击后可以删除该情绪词。
- 默认提供的情绪词不允许删除，确保核心功能完整性。

### 3.5 文本输入区域（我现在的心情页面）
功能：用户可以输入一段文字，描述自己的情绪或感受，系统根据输入内容提供鼓励或安慰的话语。

实现方式：
- 在情绪选择区域下方提供一个输入框，背景为深色，文字为浅色，与整体风格一致。
- 输入框上方可以有提示文字，如"或者，描述一下你的感受..."。

### 3.6 结果展示区域（我现在的心情页面）
功能：根据用户的情绪选择或文本输入，展示相关的鼓励或安慰话语。

展示内容：
- 情绪标题：显示所选情绪的名称，作为区域标题。
- 情绪描述：一句简短描述，解释这种情绪的含义。
- 鼓励/安慰语录：显示多条与所选情绪相关的鼓励或安慰话语，每条话语以引用块的形式呈现。
- 引用来源：每条话语后可附加来源信息（如参考图中的圣经引用）。
- **分页逻辑**：默认显示5条内容，最底部有一个"展示更多"按钮，点击后加载额外的5条内容。

### 3.7 收藏功能
功能：用户可以将喜欢的鼓励或安慰话语收藏起来，方便日后查看。

收藏方式：
- 在每条鼓励话语旁边提供一个"收藏"按钮或图标，用户点击后可以将该话语添加到收藏夹。
- 用户可以通过顶部的"我的收藏"标签页查看所有收藏的内容。

### 3.8 分享功能
功能：允许用户分享喜欢的鼓励话语，增加用户互动和网站传播性。

实现方式：
- 在每条鼓励话语和收藏内容旁边提供一个分享图标。
- 用户点击分享图标后，弹出简洁的分享选项菜单，包括：
  - 复制链接：复制该鼓励话语的直接链接
  - 微信：直接分享到微信
- 分享出去的内容包括鼓励话语全文、引用来源（如有）和网站的简短介绍。

### 3.9 我的收藏页面
功能：显示用户收藏的所有鼓励或安慰话语。

实现方式：
- 每条收藏的话语以引用块的形式展示，设计风格与主页一致。
- 每条话语旁边提供删除图标和分享图标。
- **分页逻辑**：一页显示50条内容，如内容超过50条，底部提供分页导航。

### 3.10 情绪分析功能
功能：分析和展示用户历史情绪数据，帮助用户了解自己的情绪变化趋势。

实现方式：
- 在"我现在的心情"和"我的收藏"标签页之外，添加新的"情绪分析"标签页。
- 记录用户每次选择或输入的情绪信息，包括情绪词和时间戳。
- 自动将用户的情绪归类为四个主要类别：
  - 积极情绪：喜悦、欣喜、感动、向往等
  - 消极情绪：忧伤、担忧、烦躁、无奈、迷茫等
  - 中性情绪：安宁、思念等
  - 自定义情绪：使用相似度匹配或NLP技术自动归类
- 使用数据可视化图表（如折线图、饼图）展示情绪分布和变化趋势。
- 提供不同时间范围（近一周、近一月、全部）的数据筛选。
- 显示情绪统计摘要，如最常出现的情绪、情绪变化频率等。
- 确保用户情绪数据存储在本地，保护用户隐私。
- 提供清除历史数据的选项。

### 3.11 AI自主学习功能
功能：系统通过分析用户收藏的鼓励话语，学习用户偏好的语言风格、话题和情感基调，逐步优化和个性化生成的内容。

实现方式：
- **用户行为数据收集**：记录用户收藏的内容和对应的情绪类别。
- **偏好分析算法**：分析用户收藏的话语中的共同特征，如语言风格、话题倾向和情感基调。
- **个性化内容生成**：根据分析结果，调整AI生成的鼓励话语的风格和内容，使其更符合用户偏好。
- **学习反馈循环**：用户收藏行为作为正面反馈，系统会强化生成类似风格的内容。
- **混合推荐策略**：结合基于内容流行度和个人偏好的混合推荐策略，保持一定比例的多样性。

## 4. 用户界面设计（基于参考图的暗黑风格）

### 4.1 整体风格
深邃、典雅的暗黑风格，以深蓝黑色为主色调。

配色建议：
- 主背景色：深蓝黑色（`#071323`或类似色，参考图的背景色）。
- 文字色：白色（`#FFFFFF`）或浅灰色（`#E0E0E0`）。
- 卡片背景：深蓝灰色（`#10202E`或类似色，略轻于背景）。
- 强调色：白色或浅蓝色，用于突出选中的卡片或按钮。

字体：使用现代、简洁、易读的字体，如 **Inter**、**Montserrat** 或 **Roboto**。

设计细节：
1. **标签页切换过渡效果**：添加平滑的过渡动画，如淡入淡出或滑动效果。
2. **响应式设计**：确保网页在不同设备上（桌面电脑、平板、手机）都有良好的显示效果。
3. **情绪选择的视觉反馈**：当用户将鼠标悬停在情绪卡片上时，添加轻微的放大或明亮效果。
4. **引用块设计**：保留左侧的竖线装饰和斜体文本，可为不同情绪的引用块添加微妙的颜色变化。
5. **收藏按钮设计**：使用简洁的心形或书签图标，已收藏的内容用填充图标表示。
6. **加载更多按钮**：设计为简洁的文本按钮或带有向下箭头的图标，点击后显示加载动画。

### 4.2 首页布局（我现在的心情页面）
- **页面顶部**：
  - 网站标题："Heart Verses"或"心语安慰"，大号字体，居中显示。
  - 简短描述：一段简短文字，说明网站功能，居中显示。
  - 三个标签页："我现在的心情"、"我的收藏"、"历史记录"，水平排列。当前标签页突出显示。

- **情绪选择区域**：
  - 情绪词以网格卡片形式排列，每行4-5个卡片。
  - 卡片设计为深色背景，浅色文字，选中后突出显示。
  - 末尾添加一个"+"卡片，用于添加自定义情绪词。

- **文本输入区域**：
  - 输入框设计简洁，背景为深色，文字为浅色，与整体风格一致。

- **结果展示区域**：
  - 情绪标题：所选情绪的名称，作为区域标题。
  - 情绪描述：一句简短描述，解释这种情绪的含义。
  - 鼓励/安慰语录：以引用块形式呈现，左侧有竖线装饰，引用文本为斜体或带引号，每条话语单独呈现。
  - 引用来源：每条话语右下角显示来源信息。
  - 收藏和分享按钮：每条话语旁边提供收藏和分享图标。
  - "展示更多"按钮：最底部的按钮，背景为深色，文字为浅色。

### 4.3 我的收藏页面
- 设计风格与首页保持一致。
- 每条收藏的话语以引用块形式展示，与首页的结果展示区域样式相同。
- 每条话语右侧有删除图标和分享图标。
- 一页显示50条内容，如内容超过50条，底部提供分页导航。


## 5. 其他考虑
- **隐私与数据安全**：由于工具涉及用户收藏的文字和情绪记录，建议在本地存储用户数据，确保用户隐私安全。
- **多语言支持**：如果目标用户群体包括非中文用户，可以考虑增加多语言支持。

## 6. 总结
这个网站通过优雅的暗黑风格设计和AI自主学习功能，为用户提供个性化的情绪支持和鼓励话语。用户可以通过选择情绪或输入文字，获取鼓励话语，并可以收藏、分享和回顾这些内容。网站的设计简洁而功能完整，旨在为用户提供一个宁静、舒适的情绪支持空间。
