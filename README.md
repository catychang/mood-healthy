# 心情健康网站

这是一个帮助用户管理情绪和提供心理安慰的网站。用户可以浏览心情语录、保存喜欢的内容，并进行情绪分析。

## 主要功能

- 情绪语录浏览
- 收藏喜欢的内容
- 情绪分析
- 心理安慰

## 项目结构

- `index.html` - 网站主页
- `favorites.html` - 收藏页面
- `mood-analysis.html` - 情绪分析页面
- `data.js` - 存储语录数据
- `mood-interaction.js` - 主要交互逻辑
- `mood-analysis.js` - 情绪分析功能
- `api.js` - API交互
- `icons/` - 网站图标

## 如何使用

1. 直接访问网站首页
2. 浏览情绪语录
3. 点击收藏喜欢的内容
4. 查看收藏页面
5. 使用情绪分析功能
